# Task archive

Completed task instruction files go here, one per task.

Suggested naming: `YYYY-MM-DD_<short-name>.md`
(e.g. `2026-06-17_phase-c2-matrix-from-reviewed.md`).

This keeps a record of what was requested for each Claude Code task. Move a copy
of `docs/tasks/current_task.md`'s TASK section here when a task is finished, then
clear the TASK section in `current_task.md` for the next one.
