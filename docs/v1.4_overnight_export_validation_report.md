## Method

For each directory under `runs/` (excluding the temp output folder), a
requirement source was selected by Step 6 priority:

1. `requirements_reviewed.json`
2. `requirements_clean.json`
3. `requirements_mock.json`
4. `requirements.json`

If a source existed, `scripts/export_excel.py` was run export-only, with
`responses/<case>/responses.json` passed when present. Outputs were written to
the gitignored folder `runs/_tmp_overnight_export_validation/` and validated with
`openpyxl` (workbook opens, sheet count, Compliance Matrix data-row count).

Folders without any of the four requirement-source files were recorded as
**skipped** (these are debug/worker/dry-run/empty scaffolding cases, not real RFQs).
